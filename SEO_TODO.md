# SEO 待辦清單

## P0｜先驗證，再擴張

- [ ] 在 Google Search Console 建立目前 5 組核心詞的基準：曝光、點擊、CTR、平均排名、帶來的預約／捐贈。
- [ ] 將首頁主詞定為「台北場地租借」，不要讓「文化生態圈」取代服務詞。
- [ ] 將 `/rent` 主詞定為「台北企業培訓場地／企業內訓場地」。
- [ ] 將 `/charity` 主詞定為「愛物王斷捨離二手公益」。
- [ ] 建立 `/charity/donate`，主詞定為「二手物資捐贈／捐贈規範」。
- [ ] 統一 Google 商家檔案、網站、地圖與社群的名稱、地址、電話、營業時間。

## P1｜建立搜尋頁面與轉換閉環

- [ ] 每個頁面只設定 1 個主關鍵字與 3–8 個語意相關詞。
- [ ] 場地頁補上容量、價格、設備、交通、可否佈置、實景與預約 CTA。
- [ ] 捐贈頁補上可收／不收物品、清潔分類、送達方式與聯絡入口。
- [ ] 建立首頁 → 場地 → 租借，以及公益首頁 → 捐贈規範 → Instagram 的內部連結。
- [ ] 補齊 LocalBusiness、Store、Breadcrumb、Logo、Geo、Telephone 與 OG 圖片資料。

## P2｜權威與持續成長

- [ ] 每月發佈活動案例、場地使用案例與公益成果。
- [ ] 蒐集真實 Google 評論並持續回覆。
- [ ] 取得講師、企業、公益團體與地方社群的自然引用與連結。
- [ ] 每月依 Search Console 資料淘汰低意圖詞，增加有點擊與轉換的長尾詞。

## 關鍵字策略原則

- 品牌詞：放在 H1、品牌介紹與 OG。
- 服務詞：放在 Title、頁面主標與主要 CTA。
- 地區詞：放在地址、交通、頁面標題與圖片 alt。
- 「文化生態圈」「質感活動場地」「靈動感」保留為品牌語言，不作為唯一 SEO 主詞。
